# Architecture Decision Records

This directory contains ADRs for the Personal Knowledge Graph project.

ADRs capture decisions at a point in time with explicit status. Check the status field before relying on any decision — `Proposed` means still being refined, `Accepted` means in effect, `Superseded` means replaced by a newer ADR.

## Status definitions

- **Proposed** — decision is being discussed, not yet final
- **Accepted** — decision is in effect
- **Deferred** — acknowledged but not yet decided
- **Superseded** — replaced by a newer ADR (link provided)
